Apresentação

Parabéns por avançar para a Fase 2 do PROSEL LAISA 2026! Esta etapa tem como objetivo avaliar sua capacidade de planejar e documentar o desenvolvimento de uma solução tecnológica para o setor agrícola.

Diferente de um projeto real, você irá elaborar um projeto fictício — descrevendo com clareza o passo a passo de como a solução seria construída, quais materiais ou tecnologias seriam necessários, qual metodologia seria aplicada e qual impacto se espera. A qualidade do raciocínio e da documentação é o que será avaliado.

Segmentos disponíveis — escolha 1
⚙️
Hardware Agrícola
Descreva o projeto fictício de um dispositivo físico (sensor, atuador, sistema embarcado) para resolver um problema real do campo. Inclua componentes, esquema de funcionamento e cronograma.
Eletrônica · IoT · Mecânica
💻
Software / App
Proponha um sistema, aplicativo ou script para o agronegócio. Descreva arquitetura, tecnologias, fluxo de dados, telas ou endpoints, e como seria testado.
Dev · Dados · Sistemas
🖨️
Manufatura / Design 3D
Planeje o desenvolvimento de uma peça ou estrutura fabricada (impressão 3D, corte CNC, etc.) para uso agrícola. Descreva materiais, dimensões, processo de fabricação e finalidade.
CAD · Fabricação · Design
🤖
Inteligência Artificial
Proponha uma solução de IA para o setor agrícola (visão computacional, modelos preditivos, NLP, etc.). Descreva os dados necessários, arquitetura do modelo, métricas de avaliação e aplicação.
ML · IA · Dados
Estrutura mínima do projeto

Seu PDF deve conter, no mínimo, os seguintes itens:

🎯
Problema identificado

Qual problema agrícola real você está abordando? Por que ele é relevante?

💡
Solução proposta

Descrição clara da solução fictícia, com nome, conceito e objetivos.

🔧
Materiais e tecnologias

Liste tudo que seria necessário: componentes, linguagens, ferramentas, equipamentos.

📐
Metodologia passo a passo

Como o projeto seria executado? Descreva as etapas em ordem lógica.

📅
Cronograma estimado

Quanto tempo levaria cada etapa? Apresente um cronograma realista.

🌱
Impacto esperado

Qual seria o benefício real para o produtor rural ou para o agronegócio?

Critérios de avaliação
Relevância agrícola — 25%
O problema abordado é real e relevante para o setor?
Qualidade técnica — 35%
A solução é tecnicamente coerente e bem fundamentada?
Criatividade — 25%
A abordagem é original e tem potencial de inovação?
Comunicação — 15%
O projeto está bem escrito e organizado?
